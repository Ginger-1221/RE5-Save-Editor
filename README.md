## Running Project

Clone the repo and install dependencies:

```bash
git clone ...
yarn install
```

## Starting Development

Start the app in the `dev` environment:

```bash
yarn start
```

## Packaging for Production

To package apps for portable `.exe`:

```bash
yarn package
```
